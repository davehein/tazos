NAME
        cmp - compare two files

SYNOPSIS
        cmp FILE1 FILE2

DESCRIPTION
        Compare FILE1 with FILE2.  If there is a difference between the two
        files the byte number and line number are printed.  If there is no
        difference the program will return without printing anything.
