NAME
        cal - print a calendar

SYNOPSIS
        cal [[month] year]

DESCRIPTION
        This utiliy will print a calendar for the current month if no command
        line parameters are specified.  If only the year is specified a
        calendar for all twelve months of the year will be printed.  If both a
        month and a year are specified the calendar for that month will be
        printed.
