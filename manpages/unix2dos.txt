NAME
        unix2dos - convert from from unix to dos format

SYNOPSIS
        dos2unix [FILE]

DESCRIPTION
        The unix2dos utility converts a text file from the Unix format to the
        DOS format.  A line in a DOS text file is terminated by the carriage-
        return and line-feed characters.  A Unix file is terminated only by
        the line-feed character.
